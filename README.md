# uwuifier

An uwuifier for Windows Terminal.

## Installation

[Releases](https://github.com/unami-fx/uwuifier/releases)

In the releases section of this repository, install 'uwuifier-setup.exe' from the latest release and run it. Follow the steps in the installer.

## Usage:

In a terminal, run:

```powershell
uwuify <"text to uwuify">
```
