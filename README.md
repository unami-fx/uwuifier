# uwuifier

An uwuifier for Windows.

## Installation

In the [releases](https://github.com/unami-fx/uwuifier/releases) section of this repository, install 'uwuifier-setup.exe' from the latest release and run it. Follow the steps in the installer.

## Usage:

In a terminal, run:

```powershell
uwuify <text to uwuify>
```
